Network Components

Cloud networking or physical networking

In contrast to the past, where it was a must to have a PBX or PABX and large servers to run a large-scale network, today's technology makes it possible to run your business without physical servers, and it is possible to have a mainly wireless network. This is then called Cloud Computing.

These types of networks need at least a router to connect the main elements, and the rest can be connected via wireless connection methods like Bluetooth and Wi-Fi. The main benefits of a cloud-computed network are the startup costs, which are relatively lower than those of a physically oriented network.

As the network grows and becomes more complex, these factors will increase the costs of a cloud-computed network

Physical networking has a more expensive setup cost, takes more time, and requires an expert level of troubleshooting. You have more control over the access points, especially if your servers are in-house. If there is an increase in users, you do not incur any extra costs.

Software and Harware

As already defined besides your devices like laptops desktops, printers and sacnneers we need a minimium set of equipment to set op a Cloud Computing based Network. These devices has to be Edge compatibale.
  1. Networking Hardware (On-Premises/Edge)
    1. Routers: for the connection with your ISP
    2. Switches: high-performance Ethernet (e.g., 10G/25G/100G)
    3. irewalls & Security Appliances: Next-Generation Firewalls (NGFW) or Unified Threat Management (UTM) devices
    4. Access Points (APs): Quality wireless accespoints
    5. Cabling: High-speed fiber optic or Ethernet cabling (Cat6/6a)
  2. Computing and Storage Hardware
    1. Servers: Depending on your needs
    2. Storage Devices
    3. Workstatios/Clients: laptops Desktos, Tablets etc.
  3. Cloud Networking Infrastructure (Virtual)
    1. VPC (Virtual Private Cloud)
    2. Load Balancers
    3. VPN (Virtual Private Network)
  4. Supporting Infrastructure
    1. UPS (Uninterruptible Power Supplies):For emergency power supply for your network during power outages.
    2. Environmental Control: cooling systems.
